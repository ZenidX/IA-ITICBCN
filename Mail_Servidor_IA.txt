  Especialitzat en codi. Ideal per generació i revisió de codi, documentació tècnica.

  Model lleuger. Per proves ràpides i prototipat.

COM UTILITZAR-HO?
=================

1) Via navegador: Accediu a http://192.168.1.22:8080 i ja podeu xatejar.

2) Via API amb Python:

   import requests
   response = requests.post('http://192.168.1.22:11434/api/generate', json={
       'model': 'gpt-oss:120b',
       'prompt': 'El teu prompt aquí',
       'stream': False
   })
   print(response.json()['response'])

3) Compatible amb OpenAI SDK (base_url='http://192.168.1.22:11434/v1')
